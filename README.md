# 🎙️ UniVoiceNet

UniVoiceNet is a voice authentication system that verifies user identity using **MFCC-based feature extraction** and **RNN + HMM** models for speaker recognition. It supports both **live voice recording** and **audio file uploads** for authentication, with basic anti-spoofing checks to improve security.

## ✨ Features
- 🔐 Voice-based user authentication
- 🎤 Live voice recording
- 📁 Audio file upload support
- 🧠 MFCC feature extraction
- 🤖 RNN and HMM for speaker recognition
- 🛡️ Basic anti-spoofing mechanism
- 💾 SQLite database integration
- 🌐 Flask-based web interface
